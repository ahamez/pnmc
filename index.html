Presentation

Pnmc is a small symbolic model checker for Petri nets written in C++14. It uses the libsdd library which implements Hierarchical Set Decision Diagrams.

Features

  • Symbolic generation of state spaces
  • Verification of dead states (deadlocks)
  • Verification of dead transitions
  • Verification of a subset of the reachability properties of the Model Checking Contest

Additionally, it can generate the state space of a Timed Petri nets using discrete time semantics.
