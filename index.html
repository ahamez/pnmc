Petri Net Model Checker

pnmc is a small model checker for Petri nets. It is build on top of the libsdd C++ library, which implements Hierarchical Set Decision Diagrams.

Its main focus is efficient state space generation. As such, only dead states are computed. Reachability queries are planned.
